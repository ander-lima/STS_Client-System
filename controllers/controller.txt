Camada de Controle (Controller):

    Receber requisições do usuário
    ex: clique em “Login”, envio de formulário, URL acessada.

    Processar lógica
    ex: Validação de formulário

    Decidir ações com base nos dados recebidos

    Chamar o Model para pegar ou salvar dados

    Escolher a View a ser exibida

    Passa os dados que o usuário precisa ver (ex.: mensagens de erro, resultados, listas)

💡 Resumo em uma frase:
O Controller é o “meio de campo” que conversa com o Model e a View, decide o que fazer com os dados e qual página mostrar ao usuário.
